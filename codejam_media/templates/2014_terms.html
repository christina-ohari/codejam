LGE CODE JAM 2014 CONTEST

  • 경연 시간:
    • 3월 28일(금) 13:00 ~ 3월 31일(월) 13:00 (KST 기준)

  • 경연 평가 방식:
    • 문제를 해결하면 각 문제에 배당된 점수를 획득하게 됩니다.
    • 획득한 점수로 순위가 매겨집니다.
    • 동점자의 경우 최종 Pass 한 Output을 제출한 시간으로 순위가 매겨집니다.

  • 결과 공지:
    • 결과는 코딩 전문가 collab 페이지에 소스와 함께 게시됩니다.
    • 상위 입상자는 10월에 예정된 코딩 전문가 후보자 경연에 참가할 수 있는 자격이 부여됩니다. (개별 e-mail 연락)
  • Competition Period:
    • 3.28(Fri) 13:00 ~ 3.31(Mon) 13:00 (KST Time Zone)

  • How to award:
    • After you pass each problem, you take a given score.
    • Rank is up to scores.
    • If some have same score, then the rank will be given by the order of submission.

  • {% trans "Result Announcement:" %}
    • The result will be announced in collab site with source code.
    • Participants with high score will have a chance of coding experts, who will be selected in October.
